Riemann Integral

This is a python implementation for computing the Riemann integral ∫ a b f ( x ) dx using first principles. It returns the average of the upper sum U ( f , P n ) and the lower sum L ( f , P n ) , where P n is the partition of the interval [ a , b ] with n points. (Github Link)

Lebesgue Integral

This is a python implementation for computing the Lebesgue integral ∫ f d μ in one-dimension using first principles. It returns the average of the upper Lebesgue sum U Leb n ( f ) and the lower Lebesgue sum L Leb n ( f ) computed as

U Leb n ( f ) = ∑ k = 0 n y k + 1 μ ( E k )
L Leb n ( f ) = ∑ k = 0 n y k μ ( E k ) ,

where [ y i , y i + 1 ] , i = 0 , … , n − 1 , is the partition of the range of f and μ ( E k ) is the measure of each pre-image set E k . (Github Link)
